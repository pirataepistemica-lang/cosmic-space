# Cosmic Universe Portal

Projeto Next.js "Cosmic Universe Portal" — interface com cenas 3D do universo.

Como rodar localmente

1. Instale dependências:

```bash
npm install
```

2. Rode em modo de desenvolvimento:

```bash
npm run dev
```

3. Build para produção:

```bash
npm run build
npm run start
```

Publicação no GitHub

- Criar repositório remoto no GitHub chamado `cosmic-space`.
- Adicionar `origin` e empurrar (push) para `main`.

Se quiser, posso gerar os comandos para criar o repositório remoto automaticamente usando o GitHub CLI (`gh`) — me avise se quer que eu faça isso aqui (preciso que `gh` esteja instalado e autenticado), ou crio instruções para você criar o repositório via interface web.
